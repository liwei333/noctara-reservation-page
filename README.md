# Noctara Reservation Page

Noctara 梦境观测庭内测预约静态页。

## 文件

- `index.html`：页面结构。
- `styles.css`：视觉样式。
- `assets/noctara-poster.png`：宣传主视觉。
- `assets/noctara-qr.png`：腾讯问卷预约二维码。

## 当前预约方式

这是 GitHub Pages 可直接托管的纯静态页面，预约留资使用腾讯问卷。

当前转化方式：

1. 用户打开页面。
2. 用户点击预约按钮或扫描二维码。
3. 用户进入腾讯问卷并提交预约信息。

问卷地址：

```text
https://wj.qq.com/s2/27223263/cbba/
```
